##Steps I toke ##
* 3 way to check current applications on windows
  1. Navigate your desktop
     - if an application is pre-installed it will show in your desktop
     - ex. if google was installed, you would see a google chrome icon
       
  2. Check file explorer
     - navigating to your taskbar > click file explorer > navigate to "download" folder > shows all           downloaded applications on your device
       
  3. Settings > apps
     - navigate to your taskbar > click settings > click apps > shows all apps installed
     - shows apps installed
     - shows version
     - lets you UNINSTALL applications
       
* How to install firefox
  1. Using a pre-installed browser (chrome) double-click to open application
  2. In search bar type "firefox"
  3. should show `firefox.com` > click on site
  4. will show a download prompt for windows > click download
  5. Wait for download to be complete
  6. Double click the installer.exe in the downloads in your browser
  7. Will open automatically and start installing on your computer
  8. Will be prompt a couple questions before installation (read carefully)
  9. Installed!
      
* How to check the current version and update an application (VLC media player)
  1. Navigate to VLC in desktop > right click application > click "open file location" > leads to           where the application is > Hovering over VLC in file explorer will show the current version of         the application
  2. Go to settings > apps > search for VLC > shows current version
  3. To update VLC > use pre-installed browser (chrome)
  4. in search bar type "VLC"
  5. It will show 2 different versions of VLC, make sure you know which one you downloaded
  6. When on site > install VLC again > make sure it matches your computer architecture
  7. double click installer.exe
  8. will prompt questions before updating so read carefully
  9. will update automatically after finishing prompted questions
  10. VLC now updated
      
* How to uninstall GIMP
  1. uninstalling the application in file explorer does not get rid of the application completely
  2. double-clicking the application on the desktop will allow you to remove it from the desktop view       but not deleted completely
  3. Navigate to settings > apps > find GIMP and uninstall > this will delete all its contents from         your computer
