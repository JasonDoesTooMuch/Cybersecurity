## Lab Name & Date : Installing, updating, and removing software in windows 11.2.25 ##

---
## Objective / Goal of the lab ##
Perform basic software maintenance on Windows: 
- Install Mozilla Firefox web browser 
- Update VLC Media Player to the newest version
- Unistall GIMP photo-editing tool from Virtual Machine, removing it entirely 

---
## Tools used ##
* Chrome
* File explorer
* Qwiklabs virtual machine environment (pre-configured sandbox)
* Windows OS (GUI environment)
* Settings application 


___
##Steps I toke 

### Checking Installed Applications 
1. **Desktop:** Look for application icons to confirm installation
2. **File Explorer:** Check the *Downloads* folder and programs directories
3. **Settings > Apps:** View installed applications, verions,and uninstall options
       
### Installing Mozilla Firefox 
1. Open Chrome
2. Navigate to **https://www.mozilla.org/firefox/**
3. Download the Windows installer
4. Run the `.exe` file from the Downloads folder
5. Follow installation prompts to complete setup 
      
### Updating VLC Media Player
1. Check current version through **Settings > Apps** or through File Explorer
2. Go to the official VLC website and download the newest version
3. Run the installer and follow the update prompts
4. Confirm version updated successfully 
      
### Uninstalling GIMP
1. Uninstalling the application in file explorer does not get rid of the application completely
2. Double-clicking the application on the desktop will allow you to remove it from the desktop view       but not deleted completely
3. Navigate to settings > apps > find GIMP and `uninstall` > this will delete all its contents from         your computer
